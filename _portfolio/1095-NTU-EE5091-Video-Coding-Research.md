In 2015, a joint program of MediaTek and NTUEE course EE5091 developed Intra Block Copy(IBC) algorithm for High-Efficiency Video Coding(HEVC) testing model(HM-1.0). My implementation of the Diamond search algorithm saves 70% of the time which full search takes. My implementation of the IBC predictor for video size performs the maximum compression ratio of 90%.
